Documents Required for IEPF Share Recovery

March 10, 2026 4 min read By Sharesuraksha Team

When claiming shares from the Investor Education and Protection Fund (IEPF), having the right documents is crucial. Missing or incorrect documents are the #1 reason for claim rejections. Here's a complete checklist of everything you'll need.

1. Identity Proof Documents

  • PAN Card - Mandatory for all claims
  • Aadhaar Card - For identity verification
  • Voter ID or Passport - Additional ID proof if available
  • Driving License - Can be used as address proof
Ensure all documents are self-attested and clearly legible.

2. Share-Related Documents

  • Original Share Certificates - If available (even if damaged)
  • Folio Numbers - From old share certificates
  • Dividend Warrants - Old dividend payment slips
  • Company Correspondence - Any letters from the company

3. IEPF-Specific Forms

  • Form IEPF-5 - Main claim form (we help you fill this)
  • Indemnity Bond - On appropriate stamp paper (₹100-500)
  • Advance Stamped Receipt - For acknowledging payment

4. Bank Account Proof

  • Cancelled Cheque - With pre-printed name
  • Bank Passbook Copy - First page with photo and signature
  • Bank Statement - Last 3 months (if cancelled cheque not available)

5. Demat Account Details

  • Demat Account Statement - Showing Client ID and DP ID
  • DP Details - Depository Participant name and address
  • NSDL/CDSL Client Master List - If available

6. Additional Documents (If Applicable)

For Name Changes:

  • Marriage Certificate (for married women)
  • Gazette Notification (for legal name changes)
  • Affidavit for name change

For Signature Mismatches:

  • Signature verification from bank
  • Affidavit explaining signature variation
  • Old signed documents for comparison
Pro Tip: Create a checklist and tick off each document as you gather them. We'll review everything before filing to ensure no rejections.

Document Format Requirements

  • Size: All documents should be A4 size
  • Clarity: Scans should be clear and legible
  • Color: Color scans preferred for ID proofs
  • Self-attestation: Sign across the document with date
